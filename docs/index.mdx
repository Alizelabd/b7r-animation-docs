# Animate On Scroll

**b7r-animation** is a powerful, flexible, and lightweight React animation library for creating performant scroll-based and text animations using GSAP and ScrollTrigger. Build beautiful, professional entry animations with ease.

---

## ‚ú® Key Features

- **Performance First**: Optimized animations that don't block the main thread.
- **Simple API**: Intuitive API with TypeScript support.
- **Scroll-Based**: Built on GSAP and ScrollTrigger for high-performance scroll animations.
- **Text Effects**: Easily split and animate text by characters, words, or lines.

---

## üöÄ Quick Start

### Installation

```bash
npm install gsap @gsap/react
```

### Basic Usage

```jsx
import { AnimateOnScroll, AnimationItem } from './components/animation';

function App() {
  return (
    <AnimateOnScroll>
      <AnimationItem variant="fromTop">
        <h1>Hello World</h1>
      </AnimationItem>
      <AnimationItem variant="pop">
        <p>This animates on scroll!</p>
      </AnimationItem>
    </AnimateOnScroll>
  );
}
```
