# Animate On Scroll

**b7r-animation** is a powerful, flexible, and lightweight React animation library for creating performant scroll-based and text animations using GSAP and ScrollTrigger.

Build beautiful, professional entry animations with ease. Whether you're animating elements on scroll, creating text effects, or building complex animation sequences, b7r-animation provides the tools you need.

---

## üéØ Why Choose b7r-animation?

- **Zero Configuration**: Works out of the box with sensible defaults
- **Performance First**: Optimized animations that don't block the main thread
- **Developer Friendly**: Simple, intuitive API with TypeScript support
- **Highly Customizable**: Create custom animations with full control
- **Production Ready**: Used in production applications
- **Small Bundle**: Minimal overhead, only requires GSAP

---

## ‚ú® Key Features

- **Variety of Animation Variants**: Fade, slide, pop, scale, rotate, flip, and zoom effects
- **Text Animations**: Split text into chars, words, or lines with staggered animations
- **Staggered Animations**: Control animation timing via `stagger` prop for fluid motion
- **Context-Based Architecture**: Provider pattern for flexible animation registration
- **Customizable Triggers**: Fine-tune animation timing, scroll positions, and easing
- **Scrub Support**: Create scroll-linked animations that follow scroll position
- **Debug Mode**: Visual markers to help fine-tune scroll trigger positions
- **Custom Animations**: Create your own animation presets easily

## üöÄ Quick Start

### Installation

```bash
npm install gsap @gsap/react
```

### Basic Usage

```jsx
import { AnimateOnScroll, AnimationItem } from './components/animation';

function App() {
  return (
    <AnimateOnScroll>
      <AnimationItem variant="fromTop">
        <h1>Hello World</h1>
      </AnimationItem>
      <AnimationItem variant="pop">
        <p>This animates on scroll!</p>
      </AnimationItem>
    </AnimateOnScroll>
  );
}
```
